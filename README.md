
# Portfolio

> 🚀 A sleek, professional developer portfolio to showcase your projects, skills, and experience.

## 🌐 Live Site
**[Visit Portfolio →](https://portfolio-may25.vercel.app)**

---

## 🛠️ Built With

- **Next.js (Pages Router)** – Production-grade React framework
- **React 18** – Component-based UI
- **TypeScript** – Type-safe development
- **Framer Motion** – Beautiful animations
- **Tailwind CSS / Custom Styling** – Utility-first and responsive design
- **Vercel** – Lightning-fast deployment and CI/CD

---

## 📁 Folder Structure

```
crafted-tech-visions/

```

## 📸 Features

- 🎯 **Hero, Projects, Skills, Contact Sections**

- 📱 Fully responsive across devices

- 🌗 Dark mode–ready design (optional toggle)
